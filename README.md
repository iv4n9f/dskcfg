# dskcfg

Desktop Configuration

For installation : 

cd ~ && mkdir -p Repositories && cd Repositories \n
git clone https://github.com/iv4n9f/dskcfg \n
cd dskcfg && chmod +x main.sh \n
./main.sh \n
